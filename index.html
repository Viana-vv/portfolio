BLOOM – Sistema de inscrições para eventos de ONGs

BLOOM é um site desenvolvido para simplificar o processo de inscrição em eventos de ONGs, reduzindo burocracia e melhorando a experiência dos participantes.

Minhas contribuições: Desenvolvimento completo backend e frontend, garantindo integração entre interface e banco de dados Implementação de CRUD para gerenciamento de participantes e eventos Aplicação de regras de negócio para diferentes perfis de usuários, controlando acesso e permissões

Tecnologias:PHP, HTML, CSS, MySQL, PHPMyAdmin

Marketing Paths – Plataforma para microempreendedores e pequenas empresas

Marketing Paths é uma plataforma acadêmica que auxilia microempreendedores e pequenas empresas a organizar e acompanhar suas ações de marketing, com registro de conteúdos e atividades.

Minhas contribuições: Desenvolvimento completo backend e frontend, integrando todas as funcionalidades com banco de dados Implementação de CRUD para gerenciamento de conteúdos, usuários e campanhas Aplicação de regras de negócio, definindo fluxos distintos para diferentes tipos de usuários Desenvolvimento, manutenção e programação de banco de dados via PHPMyAdmin Integração com bibliotecas externas e JavaScript, garantindo funcionalidade dinâmica Sistema responsivo para diferentes dispositivos Implementação de validação de dados e autenticação básica

Tecnologias: PHP, PHPMyAdmin, HTML, CSS, JavaScript, MySQL
